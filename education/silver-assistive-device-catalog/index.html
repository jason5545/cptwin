推薦.....類『工具書』一本用心製作的銀髮輔具目錄本(加碼版)

作者 舜英發布 2017年10月11日5 min
就學與學習

個人對輔具新知一直有高敏感度(應該已經視為個人在平常生活中會留意的嗜好)

對於家有長輩同住   但...對銀髮輔具不甚了解的我 本能的也會留心一下

明陽來村是家裡與捷運站間常往返的路  他是一家醫材輔具展示中心與醫療用品店

路過的當下發現....又放了些目錄本提供來往民眾免費索取

隨手帶了一本，回家擱置著...今天拿來仔細翻翻

哇...原來又更新許多新知  真值得是一本好用的工具書 推薦給朋友們 說這本目錄是本好用的工具書  一點也不為過

裡面以圖示、簡單易懂、如看漫畫書的一目了然

當家人是需長照又若準備出院時  這本類『工具書』可真是很貼心的選購指南

  • 所需的輔具選購、申請政府補助流程、政府可提供的服務資格、服務區域與電話或需準備資料與需求服務

製作成圖表方便查閱

  • 哪些事我們需要  似曾了解但....又不是了解很明白的事用不同顏色分類說明
  • 例如：選購輪椅需求、輪椅種類別、選購時的注意事項：(右圖中的★號 ) 座寬、座深、背靠高度....多數個廠商的各式輪椅

    

  • 或是減壓墊的選購與種類(下圖中的★號 ) 或是從右上➨可處看見補助款項適用座墊都以顏色區分清楚

    

  • 斜坡板補助項次、如何選族、規格與材質圖說範例

  • 或是電動照顧床的二馬達與三馬達是怎麼分??或是應該如何調整到最適躺姿位置嗎??  這本目錄指南圖說得簡單明瞭(下圖中的★號 ) 及容易得褥瘡部位與分級、選購

    

  • 還有最重要的轉移位省力搬概念與搬運帶、如何搬移被照顧者衛教與爬梯機種類介紹(下圖中的★號 )

     

  • 選擇助行器、拐杖及使用方式

     

  • 連浴室如廁洗澡輔具、成人紙尿片(褲)如何選購種類也介紹得仔細

        

  • 甚至氧氣機、抽痰機、噴霧機使用注意事項都有呀～～

  • 當然還有日常用品的輔具  例如：雨衣、省力筷子、可彎湯匙、圍裙、圍兜

當然目錄本裡面必定有該公司展示的各類型錄產品

但...站在家有長者有需長照服務之家屬來說

能在購買輔具時，能有一份詳細說明、有參考依據可選購指南

而非只是片面的聽從醫材用品店員推薦而盲從選購

浪費了$、且買錯輔具或是不恰當使用

更是傷了提升被照顧者生活品質、照顧者選購的心思 所以...建議有需要的朋友們

趕緊去免費索取一本來參考看看～～  謝謝窩心的老闆
