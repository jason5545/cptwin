媽媽經

為了早產兒子常困擾於看不清楚電腦上的候選字

在找到解決之道的2014.2月

寫了一篇**「電腦新注音輸入字及候選字窗字型大小之設定」**

隔了3年....在科技日新月異飛速進步下

又發現了更簡單、易上手的方法

這回是分享從螢幕顯示

分享給朋友們～

我們都知道桌面圖示除了可以利用滑鼠右鍵改變圖示大小外

這回是從『控制台』進入的

1).在控制台找到/所有控制台項目/顯示

找到/『設定自訂文字大小』去做選項  並套用

一旦按下「套用」，下次登入時就會變更(按 "套用")

按下的同時(電腦會自動關機並重新開機) 這個方式只是用於螢幕、其他圖示、圖型，當然也包含候選字的改變

而與之前的只有變更電腦候選字，有大不同

就依個人的喜好使用設定之

2**.這時電腦會自動關機並重新開機，套用新的變更(150%桌面)**

我們會發現桌面螢幕圖示變大囉～

若點入瀏覽器輸入中文或英文會發覺「候選字」也變成150%大喔～
