# 經驗分享   輔具評估去哪.....??我選擇去輔具評估中心的原因

身心障礙的朋友一定都有申請輔具的經驗
**跑來跑去的往返區公所、帶著本人或家屬的證件、印章、預約院復診時間、預約復康巴士到醫院**
**拿診斷證明書、再帶去區公所申請、等公所核可再購買....太冗長的手續呀....麻煩**
 
 
為了讓我們身心障礙者都能取得適用的輔具，促進生活自立性與健康及有更好的生活品質
這些**妥善的輔具服務，真的是生活中不可或缺的重要利器呀**~
經過多年努力，**我國於民國101年(2002)7月11日公布**
**「身心障礙者個人照顧服務辦法」、「身心障礙者輔具資源整合與研究發展及服務辦法」、「身心障礙者輔具費用補助辦法」、「身心障礙者輔具費用補助基準表」與「身心障礙者服務人員資格訓練及管理辦法」等五大重要法規**
開啟輔具服務與被服務的新紀元。
其中，在**[「身心障礙者輔具費用補助辦法」](http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=D0050060) 裡**
詳列172項輔具費用補助項目的補助對象、專業評估人員與相關評估規定、規格或功能規範、最高補助金額、最低使用年限與其他規定
並檢附25份由中央發布全國統一的輔具評估報告書。
 
**依[身心障礙者輔具費用補助辦法](http://law.moj.gov.tw/LawClass/LawContent.aspx?PCODE=D0050060) #第六條規定 ：**

```
除輔具項目不須評估或應由醫師開立診斷書者外，
```

```
主管機關自行或委託設置之輔具中心辦理評估，或由申請人依補助基準表規定至輔具服務提供單位辦理輔具評估。

也就是說有些輔具是可以由醫院的斷證明書即可送區公所申請(附上網址與項別於文章末)

但有些輔具是必須經過專業的輔具中心評估者進行評估需求，並於完成評估後十日內，將評估報告書送交申請人
```

這就牽涉到輔具評估中心
**輔具評估中心之評估員其資格需依**
**身心障礙者服務人員資格訓練及管理辦法#第14條規定者，才能晉用**
[http://www.6law.idv.tw/6law/law3/%E8%BA%AB%E5%BF%83%E9%9A%9C%E7%A4%99%E8%80%85%E6%9C%8D%E5%8B%99%E4%BA%BA%E5%93%A1%E8%B3%87%E6%A0%BC%E8%A8%93%E7%B7%B4%E5%8F%8A%E7%AE%A1%E7%90%86%E8%BE%A6%E6%B3%95.htm](http://身心障礙者服務人員資格訓練及管理辦法)
所以**評估人員必須在衛福部每年辦理輔具評估人員培訓課程進行研習，才符合資格**
這麼一來，我們可放心的在輔具評估中心進行由專業人員做評估
 
**那麼在醫院與輔具資源評估中心做評估有哪不同嗎??**
 
**個人經驗：**

- **輔具資源評估中心評估員，對輔具的了解較醫院裡的醫生、治療師熟悉**
- **輔具資源評估中心評估員，較了解個案在生活中的需求**
- **可依個人選擇方便的時間前往`，較醫院的門診時間寬裕**

**（例如：台北市合宜輔具中心，可請先去電申請輔具評估。申請後七日內治療師將於一周內主動聯繫預約評估地點與時間，評估完成後，將於十日內寄發評估報告。)**

- **輔具資源中心裡有較多不同的輔具可試用，這一點是醫院來的多(輔具是一定要試過、用過才知到哪一項適合、好用)**
- **家屬也可請廠商帶著適用的輔具，到輔具資源評估中心，請評估員一同評估輔具的適合性**
- **輔具若經輔具資源評估中心評估適用，出售以後輔具若有、保養、或問題需維修，保固期或保固期後，都可由輔具資源中心，接洽協助後續服務。**
- **★★輔具資源評估中心的評估費用比醫院拿診斷書來的便宜、方便**

**(例如：台北市合宜輔具中心，自105年2月起免收評估費用，而醫院則必須收取掛號費、評估費、診斷證明書費起碼也要$600以上)   ，這個價差實在也太大囉**

- ```
  依身心障礙者輔具費用補助辦法 #第七條規定：
  ```

  ```
  罹患嚴重疾病、行動困難、外出能力受限或有其他特殊事由經直轄市、縣（市）主管機關認有必要者
  ```

  ```
  輔具中心應指派輔具評估人員到宅進行評估，符合到宅或到機構評估之服務對象，並收交通費200元。
  ```

  ```
  (中低收及低收入戶則免)
  ```

基於以上這幾點利多需求

真心推薦

買輔具前，走一趟輔具資源評估中心準沒錯
