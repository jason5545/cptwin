# 春秋天過敏兒易引起的小兒(小腦麻兒們)咳嗽，可以居家準備的好幫手

**我家有位非常敏感的<過敏兒>**
自她**出生4個月時**就因為季節交替**引起過敏兒住院**
**在20多年前，<過敏>這詞在兒科門診還新鮮呢!**

**過敏引起的支氣管炎，肺炎是常見的併發症之一**

 
雙胞胎出生後，正因同時感冒而忙著徒手拍背時
我的摯友是位護理長，給了我一個建議
**小兒的氣管細小，何不買一台<超音波噴霧器>(如下左圖)與拍背器(如下右圖)**
在家可以加強護理
[![DSC_0846](/content/img/2014/03/DSC_0846.webp)](/content/img/2014/03/DSC_0846.webp)        [![DSC_0847](/content/img/2014/03/DSC_08471.webp)](/content/img/2014/03/DSC_08471.webp)
**一般因兒科支氣管炎住院，多以藥物與呼吸治療為先**

**呼吸治療運用水蒸霧氣吸入一段時間，把在肺部裡的濃稠的痰給稀釋時**

**利用拍擊(用拍背器)與姿勢擺位(這衛教護理就要找醫院護理師教導囉)做引流**

**讓痰的位置慢慢、稀釋的流出**

**重點是~~摯友說只要加開水，不用加任何藥物，這與一般在醫院不同**
**真是簡單又方便**
(其實也是因為當孩子發燒時，水份的補充少，所以借水蒸氣呼吸潤潤不舒服的身子，減少燒、燒、燒............)
持續的時間，與藥物共同進行
**約1星期~~10天即可改善、見效**(可從鼻子的鼻涕中檢視的到一球鼻屎)(嘔.......)
好神奇喔~
 
**所以，我家的腦麻兒們若是有支氣管炎**
**每每進馬偕兒科何啟生醫師門診時，醫師都說多多配合著拍背呼吸治療**
我們都能平安度過
**還好家裡備有這一寶，真的減少了往返急診室的舟車勞頓**

| 兒童衛教-小兒呼吸治療的護理指導 |
|  | 一、 呼吸治療原理...... |
|  | 可提供溼氣以稀釋濃稠的痰及利用重力引的方式使小孩能順利將痰排出改善呼吸情況。 |
|  | 二、治療方式...... |
|  | 1. 噴霧療法：它是一台利用超音波會噴出霧氣的機器，小孩只需做吹氣球的動作，使霧氣中的藥物或溼氣可進入深部的氣管，作用在氣管上，可稀釋濃稠的痰使易咳出達到治療的效果。 2. 姿位引流：我們可依引流部位的不同，利用枕頭協助患孩於正確的位置上，通常較常用的是將枕頭放於患孩的胸腹部的地方，讓患孩舒適的趴在枕頭上。做姿位引流的時間可在睡前、飯後一小時、或飯後二小時，以避免小孩嘔吐。 3. 胸部扣擊法：扣擊時可將手指合併彎曲杯狀，利用手腕的力量，兩手交替且規律的由下往上扣擊背部5-~10分鐘，如此痰液較易鬆落。 |
