# 168程式是個在螢幕上可拉大拉小的鍵盤介面喔!!!

呀呀呀!!!
168程式～～～是針對一個無法使用鍵盤   而又要有key-in動作    產生出來的一個程式
如:手功能不便操作指頭鍵入者..   老年人..視力須以firefox協助.......
因為是置於螢幕前    只要能把滑鼠+游標移動   就可以鍵入語詞..  關鍵字..文字編輯..
鍵盤的大小--依個人需求設定   是 free size
對於一般人    在鍵盤鍵入動作時    基本上是沒多大差異  
但是,   
在有限制能力情況下   如:受傷.. 先天..只能用滑鼠驅動時   
一般的"螢幕小鍵盤"   或是需要使用螢幕介面去處理---key-in 打字動作  
在螢幕小鍵盤無法處理的情形下～～～
168程式 是個不錯的一個方向喔!!!
